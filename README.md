# Todo List 📝

一个现代化、美观的待办事项管理应用。

![Todo List](https://img.shields.io/badge/React-18.3-purple?logo=react)
![Vite](https://img.shields.io/badge/Vite-6.0-brightgreen?logo=vite)
![Tailwind CSS](https://img.shields.io/badge/Tailwind-3.4-38bdf8?logo=tailwindcss)

## ✨ 功能特性

- 📝 添加/编辑/删除任务
- ✅ 标记任务完成状态
- 🔍 筛选视图（全部/进行中/已完成）
- 💾 本地存储自动保存
- 🎨 现代化渐变 UI 设计
- 📱 响应式布局
- ⌨️ 双击编辑任务
- 🗑️ 一键清除已完成任务

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

应用将在 http://localhost:5173 启动

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 🛠️ 技术栈

- **React 18** - UI 框架
- **Vite 6** - 构建工具
- **Tailwind CSS 3** - 样式框架
- **Lucide React** - 图标库

## 📦 部署

### Vercel 部署

1. 安装 Vercel CLI: `npm i -g vercel`
2. 运行 `vercel` 按照提示部署

### Netlify 部署

1. 运行 `npm run build`
2. 将 `dist` 文件夹拖入 Netlify
